Linux Shell 字符串替换

2023年6月23日
 | linux
 | 阅读

Shell变量替换是指在Shell脚本中使用变量来代替特定的字符串或命令。在Shell脚本中，可以使用以下几种变量替换的方式：

  1. $variable ：使用变量名来替换变量的值。

  2. ${variable} ：使用花括号来替换变量的值，效果同上，用于需要避免变量名污染的地方

  3. ${variable:-default} ：如果变量未定义，则使用默认值。非赋值

  4. ${variable:=default} ：如果变量未定义，则将其设置为默认值。赋值

  5. ${variable:+alternate} ：如果变量已定义，则使用替代值。有值，替换

  6. ${variable:?message} ：如果变量未定义，则输出错误信息。（无值，报错）

  7. ${#variable} ：获取变量的长度。

  8. ${variable%pattern} ：从变量的末尾删除指定模式的最短匹配。(非贪婪删除，右侧开始)
