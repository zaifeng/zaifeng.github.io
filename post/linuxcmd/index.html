$ tree
.
├── a
│   └── a.txt
└── b

命令使用：将a目录文件，拷贝到b目录

1

$ tar -cvf - -C a/ .|tar -xf - -C b/

命令解释：

  • 第一个- 表示 Stdout
  • -C a/ . 打包 a目录 文件，不包括目录a，注意 /与.之间的空格
  • 第二个- 表示 Stdin
  • -C b/解压到b目录

总结：- 代表标准输入 或 标准输出

       unlink file

$ rm -- -ls
$ echo $?
0
$ ls -l
a.txt

总结： -- 代表命令选项终止符（此后，所有-、–开头都不再属于选项）
