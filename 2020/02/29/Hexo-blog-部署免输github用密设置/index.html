Hexo blog部署github免密设置

Feb 29, 2020 | HexoGithub

使用Hexo创建博客并提交到github的时候，每次都需要输入用户名和密码，相信很多人都会觉得麻烦，下面是免密提交的方法

  • 生成本地密钥，将公钥id_rsa.pub拷贝至github.com个人账户下的SSH Keys 参看Centos 免密登录

  • 密钥可用性验证

  • 修改Hexo Deployment设置

PS：部分信息来源与网上，Hexo版本4.2.0 在centos 7 下实现，有问题可共同探讨，
